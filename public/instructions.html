Overview
How the Treasure Hunt Portal Works

This portal is designed to run staggered, loop-based scavenger hunts around your scout hall or camp ground. Rather than having all youth groups swarm the exact same hiding spot at once, the system calculates individual target paths based on patrol registration order.

  • No Overcrowding: Patrols start at different stations and move through the course in a continuous staggered loop.
  • Self-Guided: Youth enter 4-digit numeric PINs or text answers on a shared hall iPad to reveal their next clue.
  • Live Spectator Tracking: Leaders and parents can watch live progress and split times on a hall TV screen in real time!
Step 1
Setting Up & Swapping Game Tracks

Access the Command Centre by tapping /admin or clicking the discreet Leader Admin button in the bottom right corner of any page.

Figure 1: The Command Centre where you switch games, setup patrols, and print report sheets.
  1. Switch Active Track Map: Use the top dropdown menu to switch between existing games (e.g., Joeys-Term3, Cub Camp, or Scout Night).
  2. Create New Track Profile: Type a title (e.g. Joey Night Week 4) and an optional password, then click Add Track.
  3. Swapping Tracks Live: Changing the active track instantly updates all youth iPads and TV displays to that game profile.
Step 2
Adding & Managing Patrols

Every group needs a registered patrol record so the system can offset their route correctly.

  1. In the Patrol Setup box on the Command Centre screen, type your patrol name (e.g., Bilby Patrol, Emu Patrol, Red Team).
  2. Choose a Patrol Color Theme Banner. This color will highlight their screen header, action buttons, and progress bars!
  3. Click Save Patrol. Your added patrols appear in a list below with options to remove them if needed.
Step 3
Adding Clues & Using Magic Write AI

Scroll down in the Command Centre to access the Clue & Challenge Builder.

CRITICAL RULE: Step numbers MUST be entered sequentially starting from 1 (1, 2, 3, 4...). Do not skip numbers!
  1. Clue Step Number: Enter 1 for your first clue, 2 for your second, etc.
  2. Unlock Code: Type a 4-digit number (e.g., 1234) or a word answer (e.g., gumtree).
    *Note: A 4-digit number automatically displays a big numeric keypad on the kids' iPad, while text answers open a standard typing keyboard.
  3. Leader Cheat Sheet Note: Type the exact physical hiding spot (e.g., Taped under the fire extinguisher near the exit door).
  4. Magic Write AI Assistant: Select an age group (Joey Scouts, Cub Scouts, or Scouts) and tap Magic Write. The system will automatically convert your hiding spot into a tailored clue riddle for you!
  5. Click Save Challenge Step.
Step 4
How Scouts Interact with the App

Open the main page (/ or tap Hunt Hub) on your hall iPad or mobile device.

Figure 2: Patrol selection home screen with the Live Quest Progress Race board.

Youth select their patrol name from the top buttons. Once selected, they see their current clue, an input box for the code, and their team's colored progress bar.

Figure 3: Youth Clue Display featuring step instructions, numeric keypad, and patrol progress.
  1. Patrols tap 🚀 Start Quest! to launch onto Station 1.
  2. They read the clue riddle, find the hidden badge station in the hall, and type the unlock code into the keypad.
  3. Entering the correct code automatically moves them to the next clue!
Step 5
TV Live Tracking & Printable Run Sheets

Open /tracking on a laptop connected to a hall TV screen or projector, then tap 📺 Fullscreen TV Mode.

Figure 4: Side-by-Side TV Radar showing real-time patrol checkpoints and station split times.
  • Active Positions (Left): Highlights which station each team is currently solving without showing secret PINs or answers.
  • Station Split Times (Right): Shows live running timers for active stations and exact completion times for finished steps.

Printable Handouts & Hardcopies

In the Command Centre (/admin), under Game Controls & Handouts, you can generate instant hardcopies:

  • Print Answer Guide 📋: Prints a master cheat sheet for leaders showing all hiding spots and answer codes.
  • Print Badges ✂️: Generates printable station tokens to cut out and hide around the hall.
  • Print Split Report 📊: Generates a post-game leaderboard report showing total hunt times and station averages for every team.
